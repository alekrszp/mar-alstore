Criando o projeto com template blank
O primeiro passo é abrir o terminal ou prompt de comando e executar o comando de
criação do projeto.
O template blank cria um projeto limpo, simples e ideal para começar do zero.
Depois de criar o projeto, o aluno deve acessar a pasta do projeto para poder executá-lo.
Nesse momento, o Expo irá gerar toda a estrutura inicial da aplicação.
Para criar o projeto execute o comando abaixo:

npx create-expo-app nome-do-app --template

Selecione o template BLANK para criar um projeto expo apenas com JavaScript.
Com o projeto criado, entre na pasta do projeto e execute:

npx expo start ou npx expo start --tunnel
